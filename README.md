**Objetivo**
Desarrollar un videojuego 3D para un dispositivo móvil.

**Introducción**
Los videojuegos en 3D utilizan una geometría tridimensional, con texturas y materiales renderizados en la superficie de GameObjects para que aparezcan como los entornos sólidos, personajes y objetos que conforman el mundo del juego. Estas son las diferencias principales con un videojuego 2D. Los juegos 3D, suelen renderizar la escena con perspectiva, por lo que los objetos se ven más grandes en la pantalla cuando se acercan a la cámara.

En esta última unidad de aprendizaje desarrollarás habilidades para el desarrollo de un videojuego en 3D. ¿Estás listo para el reto?

**Contenido**
Creación de recursos de movimiento de objetos en 3D
Integración de colisiones y configuración de cámara
Dinámicas de juego y guardado de datos
Programación mecánica y creación de niveles
Flujo de juego en entornos 3D
Gestión de Iluminación y mejoras visuales
Interfaces de usuarios para videojuegos 3D
Implementación de efectos para mejora de experiencia de usuario - UX
Publicación de videojuego
